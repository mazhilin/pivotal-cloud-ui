# pivotal-cloud-ui
PivotalCloud项目基于Node构建前端UI平台

### 项目结构

```
pivotal-cloud-ui

  pivotal-blogger-ui--------------PivotalCloud云服务播客平台

  pivotal-consloe-ui--------------PivotalCloud云服务控制平台
    
  pivotal-mall-ui-----------------PivotalCloud云服务商城平台
  
  pivotal-manager-ui--------------PivotalCloud云服务管理平台
   
  pivotal-monitor-ui--------------PivotalCloud云服务监控平台

  pivotal-open-ui-----------------PivotalCloud云服务开放平台
     
  pivotal-passport-ui-------------PivotalCloud云服务登录平台

  pivotal-podcast-ui--------------PivotalCloud云服务播客平台

  pivotal-portal-ui---------------PivotalCloud云服务门户平台

  pivotal-small-ui----------------PivotalCloud云服务程序平台
```



### 构建步骤

```
安装：npm install -S
启动：npm run dev
```
